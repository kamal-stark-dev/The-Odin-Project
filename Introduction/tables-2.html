06 — Accessibility

The scope Attribute

— — —
One Piece Luffy —
Zoro —
Sanji —

The scope attribute on th tells screen readers whether a header applies to its row or column.

For headers that span multiple rows or columns, use scope="colgroup" or scope="rowgroup".

07 — Explicit Association

The id & headers Attributes

Items Sold August 2016
Clothes
Trousers Skirts Dresses
Belgium Antwerp 56 22 43
Ghent 41 17 35

How it works:

  1. Add a unique id to each th.
  2. For subheaders, add headers pointing to the parent heading's id.
  3. For each td, add headers with all related row and column ids, space-separated.

Note: This method creates very precise header–cell associations but requires significantly more markup and leaves no room for errors. The scope approach is sufficient for most tables.
